# Wallet System – GitHub Pages Demo

This repository contains a **GitHub Pages–only Digital Wallet Demo System**  
built using **HTML, CSS, and JavaScript**.

⚠️ This is a **frontend-only simulation project**.  
No real money, banking, UPI, or financial transactions are involved.

---

## Project Purpose

The purpose of this project is to:

- Demonstrate wallet UI/UX
- Test user-to-user payment flow
- Practice JavaScript logic
- Build a future server-ready structure

This project is suitable for:
- Learning
- UI/UX demo
- Logic testing
- Prototype presentation

---

## Technologies Used

- HTML5  
- JavaScript (Vanilla JS)  
- Browser LocalStorage  
- GitHub Pages  

No backend technologies are used:
- ❌ No PHP  
- ❌ No MySQL  
- ❌ No Firebase  
- ❌ No Server  

---

## Features

- One-time user registration (browser-based)
- Wallet balance management
- User-to-user payment (demo)
- Scan & Pay flow (demo)
- Mobile number payment (demo)
- Transaction history (Debit / Credit)
- 24-hour transaction limit (₹1 – ₹1,00,000)
- Payment receipt page
- Screenshot/share-ready receipt
- Hidden admin panel (demo only)
